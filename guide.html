📲 安裝到手機

1 Android（Chrome）

用 Chrome 打開：

https://onerkk.github.io/taiwan-travel/

→ 等頁面載入後，看網址列有沒有 📥 安裝圖示

→ 或點 ⋮ 選單 →「安裝應用程式」→ 桌面出現 App ✅

💡 沒出現安裝？關掉頁面等 5 分鐘再開。Chrome 要造訪 2 次才觸發。

2 iPhone / iPad（Safari）

⚠️ 必須用 Safari（Chrome 在 iOS 不支援 PWA 安裝）

  • 點底部「分享」按鈕 ⬆（方框+箭頭）
  • 找「加入主畫面」→ 點「新增」
  • 桌面出現 App ✅
💡 安裝後打開是全螢幕，沒有瀏覽器框，跟原生 App 一樣。

🔑 申請 Google API Key（完整教學）

旅遊通使用 Google API 搜尋景點。Google 每月免費 $200 美元額度，一般旅遊用完全免費不會收費。

1 註冊 / 登入 Google Cloud

打開 console.cloud.google.com

  • 用 Gmail 帳號登入
  • 第一次會要求同意服務條款，勾選同意
  • 可能要求綁信用卡 — 這是身分驗證，不會自動扣款
↓

2 建立專案

  • 點左上角「專案選擇器」→「新增專案」
  • 名稱隨意填，例如 my-travel
  • 點「建立」→ 等幾秒自動切換到新專案
↓

3 啟用 3 個 API（最重要！）

左邊選單 →「API 和服務」→ 點上方「＋ 啟用 API 和服務」

搜尋並啟用這 3 個：

① Places API (New) — 搜尋景點餐廳住宿

  • 搜尋 Places API → 找 「Places API (New)」（有 New 字樣的）
  • 點進去 → 點藍色「啟用」

✅ Places API (New)

② Maps JavaScript API — 互動地圖

  • 搜尋 Maps JavaScript → 點「啟用」

✅ Maps JavaScript API

③ Geocoding API — 自動定位你的位置

  • 搜尋 Geocoding → 點「啟用」

✅ Geocoding API

⚠️ 三個都要啟用！少一個對應功能就不能用。
↓

4 建立 API 金鑰

  • 左邊選單 →「API 和服務」→「憑證」
  • 上方點「＋ 建立憑證」→ 選「API 金鑰」
  • 自動產生一串 AIzaSy... 開頭的金鑰
  • 點「複製」
↓

5 設定金鑰安全限制（建議）

在「憑證」頁面點你的 API 金鑰名稱：

  • 「應用程式限制」→ 選「HTTP 參照網址」
  • 新增：https://onerkk.github.io/*
  • 「API 限制」→ 選「不限制金鑰」
  • 點「儲存」
⚠️ 如果照片不顯示，把「API 限制」改成「不限制金鑰」通常就解決了。
↓

6 貼入旅遊通開始使用

  • 打開旅遊通 → 貼上金鑰 → 點「🔑 開始旅行」
  • 驗證成功就進入國家選擇頁 🎉
💡 旅伴同步：把同一組 Key 分享給旅伴 → 打卡、位置自動同步！

❓ 常見問題

照片沒有顯示？

  • 確認「Places API (New)」已啟用（不是舊版）
  • 到「憑證」→ API Key →「API 限制」選「不限制金鑰」
  • HTTP 參照網址加上 https://onerkk.github.io/*

地圖沒出現？

確認「Maps JavaScript API」已啟用，等 1-2 分鐘生效。

會被收費嗎？

Google 每月免費 $200（約 5,000 次搜尋），正常旅遊完全不會超過。可到「帳單」→「預算與快訊」設花費上限。

打卡按不了？

需要：① 登入 Google 帳號 ② 開啟定位 ③ 在景點 1km 內

⚡ 智慧行程規劃

自動推薦（推薦！）：選天數+交通 → 按「⚡ 自動推薦」→ 自動搜尋最佳景點三餐住宿
